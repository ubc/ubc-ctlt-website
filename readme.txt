1. Activate the plugin through the 'Plugins' menu in WordPress

= Banner images, how do they work? =

Banner images will only show if the images used as the feature image is set to the correct width and height. To do this go to Dashboard > Appearances > Theme Options > CTLT website Options and find Banner Settings.
